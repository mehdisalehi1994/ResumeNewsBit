یک صفحه وب استاتیک با موضوع اخبار بازار ارز دیجیتال است که به‌عنوان دومین پروژه من در مسیر یادگیری فرانت‌اند توسعه یافته است. این پروژه با HTML و CSS خالص پیاده‌سازی شده و تمامی محتوای آن به‌صورت ثابت در کد طراحی شده است. طراحی کاملاً واکنش‌گرا و Mobile-First بوده و شامل منوی برگری برای موبایل، کارت‌های خبری، لینک‌ها و دکمه‌ها با تعاملات CSS و Hover Effect است. ساختار سایت بخش‌بندی‌شده شامل اخبار برجسته، Trending News، Popular Posts و فوتر است و تجربه کاربری روان و استانداردی را ارائه می‌دهد. این پروژه توانایی من در طراحی صفحات استاتیک، واکنش‌گرا و ایجاد رابط کاربری تعاملی بدون JavaScript را به خوبی نشان می‌دهد.
