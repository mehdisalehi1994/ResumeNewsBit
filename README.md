معرفی پروژه – صفحه استاتیک اخبار بازار ارز دیجیتال (ResumeNewsBit)

این پروژه یک صفحه وب استاتیک با موضوع اخبار و محتواهای مرتبط با بازار ارزهای دیجیتال است که دومین پروژه من در مسیر یادگیری فرانت‌اند محسوب می‌شود.

هدف اصلی پروژه، تمرین چیدمان محتوایی، طراحی رابط کاربری جذاب و ایجاد تجربه کاربری تعاملی و واکنش‌گرا بوده است.

ویژگی‌ها و پیاده‌سازی فنی

توسعه با HTML و CSS خالص بدون فریم‌ورک، برای تسلط کامل روی ساختار و استایل‌دهی.

طراحی کاملاً استاتیک
محتوا به‌صورت ثابت داخل کد پیاده‌سازی شده و بدون وابستگی به بک‌اند یا API است.

ریسپانسیو حرفه‌ای و Mobile-First
صفحه در موبایل، تبلت و دسکتاپ به‌صورت کامل تطبیق داده شده است و عناصر و چیدمان‌ها با توجه به اندازه صفحه تغییر می‌کنند.

منوی برگری (Hamburger Menu) برای سایزهای کوچک
برای بهبود تجربه کاربری در موبایل، ناوبری سایت در حالت کوچک به صورت برگری درآمده و دسترسی سریع به بخش‌ها را فراهم می‌کند.

استفاده از Hover Effects و تعاملات CSS
برای لینک‌ها، کارت‌های خبری و دکمه‌ها، بدون نیاز به JavaScript.

ساختار بخش‌بندی‌شده مشابه سایت‌های خبری کریپتو
شامل اخبار برجسته، Trending News، Popular Posts و فوتر.

جایگاه پروژه در مسیر رشد

این پروژه نشان‌دهنده پیشرفت قابل توجه در درک طراحی ریسپانسیو و UX حرفه‌ای نسبت به پروژه اول است.
اضافه شدن منوی برگری و تعاملات Hover به‌صورت ملموس، نشان‌دهنده حرکت از طراحی صرفاً صفحه‌ای به سمت تجربه کاربری واقعی و استانداردهای وب مدرن است.
